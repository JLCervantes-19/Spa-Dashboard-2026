¿Cómo funciona?

• Un administrador tiene acceso completo a este panel: empleadas, reservas, servicios, clientes, configuración y testimonios.

• Al crear un admin se genera una cuenta en Supabase Auth y se registra en la tabla admin_users.

• No puedes eliminarte a ti mismo para evitar quedar sin acceso.

• Para agregar admins manualmente desde Supabase revisa el archivo INSTRUCCIONES-AGREGAR-ADMIN.md en la carpeta del proyecto.
